## üìù Deskripsi Percobaan 2

Percobaan ini berfokus pada simulasi dan konfigurasi jaringan dasar menggunakan simulator **Cisco Packet Tracer**. 

Tujuan utama dari praktikum ini adalah untuk membangun sebuah topologi jaringan sederhana, melakukan konfigurasi alamat IP (IP addressing) pada perangkat *end-device* (seperti PC), dan memverifikasi konektivitas antar perangkat tersebut menggunakan perintah **PING**.
