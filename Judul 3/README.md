## üìù Deskripsi Percobaan 3

Percobaan ini berfokus pada simulasi dan konfigurasi **VLAN (Virtual LAN)** dan **Trunking** menggunakan simulator Cisco Packet Tracer.

Tujuan utama dari praktikum ini adalah untuk memahami bagaimana VLAN dapat digunakan untuk meningkatkan performa jaringan dengan memisahkan domain broadcast Layer 2 yang besar menjadi lebih kecil, serta sebagai langkah keamanan.

Praktikum ini mencakup:
* Pembuatan VLAN dan penetapan port switch ke VLAN tertentu.
* Pemeliharaan database VLAN dan penetapan port.
* Konfigurasi **trunk 802.1Q** antar switch untuk memungkinkan lalu lintas dari beberapa VLAN melintasi satu tautan (link) fisik.
